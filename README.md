# Portfólio Pessoal – Maria Eduarda Hafner

## Descrição

Este projeto consiste no desenvolvimento do meu portfólio pessoal, criado para apresentar minha formação, trajetória acadêmica e objetivos profissionais na área de tecnologia.

O projeto foi desenvolvido como parte da disciplina **Aplicações Web e Móveis**, do curso de **Engenharia de Software**, tendo como proposta a criação de um site completo utilizando HTML, CSS e JavaScript.

Além de atender aos requisitos da disciplina, o projeto demonstra conhecimentos em desenvolvimento front-end, responsividade, manipulação do DOM com JavaScript e organização de interfaces utilizando HTML e CSS.

O layout foi desenvolvido com identidade visual própria, priorizando uma experiência simples, intuitiva e adaptável a diferentes tamanhos de tela.

**Acesse o projeto:**
https://mariahafner.github.io/portfoliomariahafner/

---

## Objetivos do projeto

Este projeto foi desenvolvido como parte de uma atividade da disciplina **Aplicações Web e Móveis**, do curso de **Engenharia de Software**, e também integra meu portfólio profissional.

Seus principais objetivos foram:

* desenvolver uma aplicação web utilizando HTML, CSS e JavaScript;
* aplicar conceitos de estruturação semântica, estilização e responsividade;
* praticar manipulação do DOM e eventos com JavaScript;
* desenvolver uma interface com identidade visual própria;
* apresentar minha formação, habilidades e objetivos profissionais em um portfólio online.

---

## Funcionalidades

* Navegação por seções utilizando menu fixo.
* Menu responsivo para dispositivos móveis.
* Seção de apresentação pessoal.
* Exibição da formação acadêmica em formato de cards.
* Área destinada aos projetos desenvolvidos.
* Vídeo incorporado apresentando o processo de desenvolvimento do site.
* Formulário de contato com validação dos campos.
* Integração com mailto, permitindo abrir o cliente de e-mail do usuário com a mensagem preenchida automaticamente.
* Layout responsivo para desktop e dispositivos móveis.
